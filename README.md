**Terminal Commands**
---
    mkdir 'folder name' - Make Directory

    cd 'file name' - change Directory

    ls -a - List all Items

    ls - list the contents of a specific folder

    touch 'file name' - Create new files

    -rf - Delete Folders

    ./ - Current working directory
    
    ../ Parent Directory
